description:

The MatrixLlong class manages a matrix of Llong objects. It is derived from the MMatrix class and inherits the commonly used mathematical, logical, linear algebra and i/o methods from it. A MatrixLlong object can read itself from (or write itself to) a Sof file. It includes the memory management methods which use the MemoryManager class to allocate or delete memory from the heap.

examples:

//

//

//

Sof tmp_file;

// open the file in write mode       
//

// write the matrix object to the Sof file
//       
val0.write(tmp_file, (long)0);

// close the file
//       
tmp_file.close();
