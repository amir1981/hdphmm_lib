description:

The MatrixUllong class manages a matrix of Ullong objects. It is derived from the MMatrix class and inherits the commonly used mathematical, logical, linear algebra and i/o methods from it. A MatrixUllong object can read itself from (or write itself to) a Sof file. It includes the memory management methods which use the MemoryManager class to allocate or delete memory from the heap.
