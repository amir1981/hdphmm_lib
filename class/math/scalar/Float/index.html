description:

The Float class manages 32-bit float numbers. It is derived from the MScalar class and inherits commonly-used mathematical, logical, DSP and i/o methods from it. A Float object can read itself from (or write itself to) a Sof file. It includes the memory management methods which use the MemoryManager class to allocate or delete memory from the heap.

 examples:

  • This example shows how to write a Float object into a Sof file: 
    Float val0(5.0);

Sof tmp_file;
tmp_file.open(tmp_filename, File::WRITE_ONLY, File::TEXT);
       
val0.write(tmp_file, (long)0);
tmp_file.close();
  • This example shows how to generate a random number:

// Gaussian random number generation
//
float mean = 5.0;

value1 = rand_num.grand(mean, std_dev);
 notes:

  • See the notes section of MScalar.

  • The Float class defines the bor, band, bxor, brs, bls, bcmpl, mod, operator%= and operator% integer methods as private to block the templatized functions in MScalar. So if a user attempts to use Float::mod() they will get a compilation error rather than a runtime error.
