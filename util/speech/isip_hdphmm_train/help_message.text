L"name: isip_hdphmm_train\nsynopsis: isip_hdphmm_train -params_file params.sof -train_file example_db.sof -itr 100,200 -init_itr 50 -burn_in 100  -structure_type LEFT_TO_ROGHT -emission_type HDP -kz 5 -output_mode TEXT -hmm_params 10,1,10,1,100,1 -emission_params 10,0,10,1,10,.1,5 -raw_model raw_model.text [-load_model saved_model.text] [-verbose none] [-debug_level none] [-extract 0][-final_model final_model.text] [-log_likelihood heldout_db.sof]\ndescription: This command can train(inference) an HDPHMM. It also supports two other modes to extract the final model and to calculate the log_likelihood of some observation.\n\nexample (train): isip_hdphmm_train -params_file params.sof -train_file example_db.sof\n\nexample(extract final model): isip_hdphmm_train -extract 1 -params_file params.sof -raw_model  raw_model.text   -final_model final_model.text\n\nexample(calculate log_likelihood): isip_hdphmm_train -final_model final_model.text  -log_likelihood example_db.sof\n\noptions:\n\n-train_file: database contain the training examples.\n-itr: number of iterations. If a vector is specified the utility save the raw model at points specified in the vector.\n-init_itr: number of iterations to initilize the algorithm (used for LEFT_TO_RIGHT model.)\n-burn_in: this indicates when utility starts accumulating the model parameters.\n-structure_type: ERGODIC/LEFT_TO_RIGHT/STRICTLY_LEFT_TO_RIGHT (default ERGODIC)\n-emission_type: HDP/DPM\n-kz: maximum number of states\n-output_mode TEXT/BINARY\n-hmm_params: hyper parameters for HMM. This is a vector of length 6. For more information refer to ???.\n-emission_params: Parameters and hyperparameters for emission models. For DPM, this is a vector of length 6 and for HDP this is a vector of length 8. The first item is the maximum number of components in the mixture model. The second item determine what kind of prioir is used (0: NIW, 1: IW-N, 2: IW-N-with-tied-covariance) and remaining are the hyper-parameters. For more info refer to ???\n-raw_model: name of the file to save or load the raw model.\n-final_model: name of the file to save or load the final model. If specified during the training the final model is also saved.-extract: if 0 it indicates the training mode and if 1 indicates extracting mode(extracting the final model from raw model without training.) (default 0)\n-log_likelihood: This option is for the third mode of opertation which use final model as input and calculate the log likelihood of the input.\n-verbosity: verbosity level (default = NONE)\n-debug_level: debug level (default = NONE)\narguments: none\n\nman page: $ISIP/util/speech/isip_hdphmm_train/index.html\n\n*  More \ndetails on these can be found in the tutorial.\n\n For further info look at  isip_hdphmm_train.cc"
